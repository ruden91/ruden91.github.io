# FERuden

A [GatsbyJS](https://www.gatsbyjs.org/) Static Web Page <br /><br />

## Description

React 기반의 GatsbyJS 를 활용한 Static Web Page Blog 개발

## Features:

* React
  * SEO (React Helmet)
  * Styled-Components
* Contentful (content API)
* Disqus Comment System
* PWA (manifes.json, offline support, favicons)
* Image lazy loading
